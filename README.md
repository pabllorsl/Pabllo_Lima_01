<h2 align="left">
Descrição:
</h2>
> Você foi contratada(o) para implementar o código de um aplicativo para o jogo Pokemon criado e comercializado pela Nintendo. Você deve criar representações para os Pokémons, uma pokeagenda e os treinadores. Nesse aplicativo você pode capturar pokémons, verificar o poder de ataque deles e comparar seu poder de ataque com o de outros treinadores pokémons. Use a classe de testes para guiar sua implementação, verificando se o teste após a implementação de cada passo descrito no roteiro.
